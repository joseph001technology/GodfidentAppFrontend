# Godfident App Frontend

A Flutter-based Bible study and spiritual growth application.

 
## Setup

flutter pub get
flutter run

## Features

- JWT Authentication
- Bible Reading
- Bookmarks & Notes
- Prayer Journal
- Reading Plans
- Daily Devotionals
- AI Bible Assistant
- Reading Analytics
- Push Notifications

 
## Notes

The backend is hosted on Render and may take a few moments to wake up after inactivity.

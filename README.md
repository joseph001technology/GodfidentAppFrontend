# Godfident Flutter App

Personal Bible study & spiritual growth app.

## Backend
Production API: https://godfidentappbackend.onrender.com

## Setup

### 1. Install Flutter
https://flutter.dev/docs/get-started/install

### 2. Get dependencies
```bash
flutter pub get
```

### 3. Run
```bash
# Android
flutter run

# iOS
flutter run -d ios

# Chrome (web, for testing)
flutter run -d chrome
```

## Project Structure

```
lib/
├── main.dart               # Entry point
├── app.dart                # MaterialApp.router
├── core/
│   ├── constants.dart      # Base URL, keys
│   ├── theme.dart          # Dark navy + gold theme
│   ├── dio_client.dart     # HTTP + JWT auto-refresh
│   ├── secure_storage.dart # Token storage
│   └── router.dart         # GoRouter + auth guard
├── models/                 # Dart data classes
├── repositories/           # All API calls
├── providers/              # Riverpod state
└── screens/                # UI screens
    ├── auth/
    ├── home/
    ├── bible/
    ├── devotionals/
    ├── reading_plans/
    ├── prayer/
    ├── ai/
    ├── analytics/
    ├── notifications/
    └── settings/
```

## Key Features
- JWT auth with auto-refresh on 401
- Bible reading with bookmarks, highlights, notes
- AI Bible assistant (explain verse, topic study, character study)
- Prayer journal with answered prayer tracking
- Reading plans with streak tracking
- Daily devotionals
- Analytics dashboard with reading heatmap
- Dark navy + gold theme

## API Endpoints Used
See https://godfidentappbackend.onrender.com/api/ for the full interactive reference.

## Notes
- First load on Render may be slow (free tier cold start ~30s)
- AI responses are generated by Claude — always verify with Scripture
- flutter_secure_storage requires Android minSdk 18+
